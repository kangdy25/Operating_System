## 🎯 학습 목표 🎯

운영체제에서 제공하는 서비스를 사용하는 능력을 배양한다. 
운영체제의 구조, 프로세스, 프로세스 동기화, 교착상태, 메모리, 파일시스템등에 대하여 학습한다.
